# Paperify Access Guide

## Overview
Paper generation access is unrestricted.

- No demo paper cap
- No monthly paper cap
- No book-lock restriction
- No forced upgrade prompt after usage
- No block for plan limits

## Access Policy
All users can generate papers without limit and continue using the system without usage-based blocking.

## Plan Display
Plan cards can still be shown for pricing/reference, but they do not enforce generation limits.

## Subscription Behavior
- Access does not stop because of paper count.
- Access does not stop because of book selection limits.
- Access is not blocked by demo usage thresholds.

## Payment Flow (Informational)
If payment collection is used, treat it as billing/record-keeping only, not as a hard gate for generation limits.
